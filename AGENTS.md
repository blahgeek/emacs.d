# Agent Notes

## Package Source & Documentation

When working with Emacs packages in this configuration, the full package source code is available at:

```
straight/repos/<package-name>/
```

This directory contains:
- **README** (or README.md, README.org, etc.) - Package documentation
- **Full source code** - Elisp files for reading implementation details

For example, hydra documentation and source is at `straight/repos/hydra/`.

This is useful for understanding:
- Package-specific syntax and macros
- Configuration options
- API documentation
- Usage examples
- Implementation details when documentation is insufficient
